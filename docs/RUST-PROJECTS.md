# Проекты для игры Rust

Автоматизация получения ежедневных бонусов на серверах Rust.

## MagicRust.gg - Бесплатный ежедневный кейс

**Домен:** `magicrust.gg`
**URL:** https://magicrust.gg/ru
**Частота:** Каждые 10 часов

### Что делает

Автоматически открывает бесплатный ежедневный кейс на сайте MagicRust.gg:

1. Переключается на вкладку "Modded"
2. Находит бесплатный кейс
3. Открывает модальное окно кейса
4. Нажимает кнопку "Открыть кейс"
5. Получает награду из кейса

### Требования для работы

#### 1. Авторизация через Steam

- При первом запуске расширение автоматически перенаправит на страницу авторизации Steam
- После успешной авторизации скрипт будет работать автоматически

#### 2. Настройка проекта

Добавьте проект в расширение:

```
URL: https://magicrust.gg/ru
```

#### 3. Разрешения

Расширение автоматически запросит следующие разрешения:
- `https://*.magicrust.gg/*` - доступ к сайту MagicRust
- `*://*.steamcommunity.com/*` - авторизация через Steam

### Файлы проекта

- **scripts/magicrust.gg.js** - основной скрипт автоматизации
- **projects.js** (строка 1386) - конфигурация проекта

## BloodRust.com - Колесо фортуны и Промокоды

**Домен:** `bloodrust.com`
**URL:** https://bloodrust.com

### Две задачи в одном проекте

BloodRust поддерживает два типа автоматизации:

#### 1. Колесо фортуны (Wheel Spin)

**URL:** https://bloodrust.com/wheel
**Частота:** Каждые 24 часа

**Что делает:**
1. Находит кнопку "Крутить" на странице колеса
2. Нажимает на кнопку для запуска вращения
3. Ожидает результата
4. Автоматически определяет время до следующего вращения

#### 2. Активация промокода (Promo Code)

**URL:** https://bloodrust.com/home
**Частота:** Каждые 6 часов (при наличии вайпа)

**Что делает (полностью автоматически):**

1. **Background Script** (работает в фоне, без открытия вкладки):
   - Проверяет календарь вайпов на сервере
   - Ищет вайп в ближайшие 24 часа (прошедший или предстоящий)
   - Парсит Telegram канал [@bloodrust](https://t.me/bloodrust)
   - Находит самый свежий промокод по ID сообщения
   - Проверяет, не был ли промокод уже использован
   - Сохраняет последние 50 использованных промокодов

2. **Content Script** (работает на странице):
   - Получает промокод от background script
   - Нажимает кнопку "BC 90" для открытия модального окна
   - Переключается на вкладку "Промокод"
   - Вводит промокод
   - Нажимает "Применить"
   - Проверяет результат (успех/ошибка)

3. **Умное расписание:**
   - Если промокод уже использован → откладывает до следующего вайпа
   - Если вайпов нет → проверяет каждый день в 00:00
   - Добавляет +30 секунд к времени вайпа (чтобы промокод успел появиться)

### Требования для работы

#### 1. Авторизация через Steam

- При первом запуске на обеих страницах (wheel и home)
- Автоматическое перенаправление на Steam

#### 2. Настройка проектов

Добавьте **ОБА** URL в расширение:

```
1. https://bloodrust.com/wheel    - для колеса фортуны
2. https://bloodrust.com/home     - для промокодов
```

Система автоматически определит тип задачи по URL.

#### 3. Разрешения

Расширение автоматически запросит следующие разрешения:
- `https://*.bloodrust.com/*` - доступ к сайту BloodRust
- `*://*.steamcommunity.com/*` - авторизация через Steam
- `https://*.t.me/*` - парсинг Telegram канала для промокодов
- `https://*.telegram.org/*` - дополнительный доступ к Telegram

### Файлы проекта

#### Колесо фортуны (Wheel Spin)
- **scripts/bloodrust.com.js** - скрипт вращения колеса
  - Проверяет URL: выполняется только на `/wheel`, игнорирует `/home`

#### Промокоды (Promo Code)
- **scripts/bloodrust.com_silentvote.js** - background script для парсинга Telegram и поиска промокодов
  - Работает в Service Worker
  - Не открывает вкладку браузера
  - Выполняется только для задач с `id === 'promo code'`

- **scripts/bloodrust.com-promo.js** - content script для активации промокода на сайте
  - Получает промокод от background script
  - Автоматически заполняет и отправляет форму

#### Конфигурация
- **projects.js** (строка 1398) - основной проект `bloodrust.com`
  - Определяет тип задачи по URL path
  - `/home` → promo code (с silent vote)
  - `/wheel` → wheel spin (без silent vote)

- **background.js** (строка 12) - загрузка `bloodrust.com_silentvote.js`

### Отдельный проект bloodrust.com-promo (опциональный)

Существует также отдельный проект `bloodrust.com-promo` (строка 1417 в projects.js), который дублирует функционал промокодов. Его можно использовать для тестирования или как резервный вариант.

## Устранение неполадок

### MagicRust

**Ошибка: "Modded button not found"**
- Обновите страницу
- Проверьте, что вы на странице https://magicrust.gg/ru

**Ошибка: "Free case button not found"**
- Возможно, бесплатный кейс временно недоступен
- Проверьте авторизацию через Steam

**Сообщение: "доступен каждые 10 часов"**
- Это нормально - расширение автоматически повторит попытку через 10 часов

### BloodRust - Колесо

**Ошибка: "Spin button not found"**
- Убедитесь, что вы на https://bloodrust.com/wheel
- Обновите страницу
- Проверьте авторизацию

**Кнопка заблокирована**
- Расширение автоматически прочитает оставшееся время
- Следующая попытка будет через указанное время + 1 минута

### BloodRust - Промокоды

**Ошибка: "Промокод не найден в Telegram канале @bloodrust"**
- Telegram канал временно недоступен
- Промокод еще не опубликован
- Расширение автоматически повторит попытку

**Сообщение: "Вайпов в ближайшие 24 часа не найдено"**
- Это нормально - расширение проверит снова завтра в 00:00
- Промокоды появляются только во время вайпов

**Промокод уже был использован**
- Расширение автоматически отложит до следующего вайпа
- История последних 50 промокодов сохраняется

**Ошибка: "BC button not found" / "Apply button not found"**
- Обновите страницу https://bloodrust.com/home
- Проверьте авторизацию через Steam
- Интерфейс сайта мог измениться - сообщите разработчику

## Логи и отладка

### MagicRust
Все логи в консоли начинаются с префикса `[bloodrust.com-promo]`

### BloodRust
- Колесо: логи без специального префикса
- Промокоды (background): `[bloodrust.com-silent]`
- Промокоды (content): `[bloodrust.com-promo]`

### Полезные логи BloodRust (промокоды)

```
✓ Найден прошедший вайп: [дата]      - вайп найден в прошлом (в пределах 24 ч)
⏳ Найден предстоящий вайп: [дата]    - вайп будет в будущем (в пределах 24 ч)
✓ Выбран самый свежий промокод: [код] - промокод успешно найден
✓ Промокод применён: [код]            - промокод успешно активирован
```

## Технические детали

### MagicRust технология
- Простой content script
- Взаимодействие с DOM элементами
- Обработка модальных окон
- Парсинг notyf уведомлений

### BloodRust технология

#### Колесо фортуны
- Content script с проверкой URL
- Парсинг времени кулдауна из HTML
- Автоматическое определение следующей попытки

#### Промокоды (Silent Vote архитектура)
- **Service Worker** для фоновой работы
- **Fetch API** для получения данных календаря и Telegram
- **Regex парсинг** HTML без DOM (Service Worker не имеет доступа к DOM)
- **IndexedDB** для хранения истории промокодов
- **Message Passing** между background и content scripts

#### Уникальные особенности BloodRust промокодов:
1. Работает полностью в фоне (silent vote)
2. Не открывает лишние вкладки
3. Парсит Telegram без API (через веб-версию)
4. Определяет новизну промокода по message ID
5. Умное планирование на основе вайпов
6. Предотвращение дублирования промокодов

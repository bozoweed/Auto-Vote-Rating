# Проекты для игры Rust

Автоматизация получения ежедневных бонусов на серверах Rust.

## Установка расширения

1. Скачайте архив с расширением
2. Распакуйте архив в удобное место на компьютере
3. Откройте браузер Chrome
4. Перейдите на страницу `chrome://extensions/`
5. Включите "Режим разработчика" (Developer mode) в правом верхнем углу
6. Нажмите "Загрузить распакованное расширение" (Load unpacked)
7. Выберите папку с распакованным расширением
8. Расширение установлено и готово к работе!

## Требования

- Steam аккаунт с включенными cookies
- Chrome-браузер версии 105.0 или выше
- Авторизация через Steam на сайтах MagicRust и BloodRust

---

## MagicRust.gg - Бесплатный ежедневный кейс

**Что делает:**
Автоматически открывает бесплатный ежедневный кейс каждые 10 часов

**Какой URL добавить в расширение:**
```
https://magicrust.gg/ru
```

**Как это работает:**
1. Переключается на вкладку "Modded"
2. Находит бесплатный кейс
3. Открывает и получает награду
4. Повторяет каждые 10 часов

---

## BloodRust.com - Колесо фортуны

**Что делает:**
Автоматически крутит колесо фортуны каждые 24 часа

**Какой URL добавить в расширение:**
```
https://bloodrust.com/wheel
```

**Как это работает:**
1. Находит кнопку "Крутить"
2. Нажимает и ожидает результата
3. Автоматически определяет время до следующего вращения
4. Повторяет каждые 24 часа

---

## BloodRust.com - Автоактивация промокодов

**Что делает:**
Полностью автоматически активирует промокоды после вайпов

**Какой URL добавить в расширение:**
```
https://bloodrust.com/home
```

**Как это работает (полный автомат):**
1. Работает в фоне, вкладку открывать не нужно
2. Проверяет календарь вайпов каждый день
3. Ищет вайп в ближайшие 24 часа
4. Парсит Telegram канал @bloodrust
5. Находит самый свежий промокод
6. Автоматически активирует его на сайте
7. Запоминает использованные промокоды (не дублирует)
8. Если промокод уже использован - ждёт следующего вайпа
9. Если вайпов нет - проверяет снова завтра в 00:00

**Важно:**
- Расширение запросит доступ к Telegram для автоматического поиска промокодов
- Всё работает в фоне, вам ничего делать не нужно
- История последних 50 промокодов сохраняется

## Быстрая настройка - Что добавить

### Если хотите всё сразу, добавьте эти 3 URL:

```
1. https://magicrust.gg/ru          - бесплатный кейс каждые 10 часов

```

После добавления:
1. Откройте каждый URL в браузере
2. Авторизуйтесь через Steam
3. Всё! Расширение начнёт работать автоматически

## Частые вопросы

**Нужно ли держать браузер открытым?**
- Да, браузер должен быть запущен
- Вкладки открывать не обязательно - расширение работает в фоне
